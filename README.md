# VibeCheck (demo)

An interactive MVP prototype for **VibeCheck**, an AI dating concept focused on helping serious daters get from match to a real date faster, without endless swiping.

- **`index.html`** is the main demo (dark "After Hours" build).
- **`classic.html`** is an alternate light, phone-style build.

Each is a single self-contained HTML file. Mock data only, no backend, no login. Open in any modern browser.

The full flow: AI intake → curated weekly matches → a 90-second voice "vibe check" → straight to scheduling a real date. The admin dashboard shows the core result: the voice check nearly doubles match-to-date conversion and cuts ghosting in half.
